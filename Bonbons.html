Site Web Marchand de Bonbons

Objectif :


Le site web marchand de bonbons est une plateforme en ligne permettant aux utilisateurs d'acheter des bonbons et à l'administrateur du site de gérer le catalogue de produits. Le site offre une interface conviviale pour la navigation des utilisateurs et une interface d'administration sécurisée pour la gestion des produits.


Langages :


PHP, HTML, et CSS


Base de donnée :


MySQL


Sécurité :


Utilisation de techniques de sécurisation des données telles que le hachage des mots de passe et la prévention des attaques par injection SQL.



Fonctionnalités pour les Utilisateurs :


  1. Catalogue de Produits : Affichage des bonbons disponibles avec leurs descriptions, prix et images.




  2. Recherche de Produits : Barre de recherche permettant aux utilisateurs de rechercher des bonbons par nom, type ou ingrédient.






  3. Panier d'Achat : Permet aux utilisateurs d'ajouter des bonbons à leur panier et de pouvoir accéder aux accès disponibles.



Fonctionnalités pour les Administrateurs :


  1. Interface d'Administration : Accès sécurisé à une interface d'administration pour gérer les produits.



  2. Accueil Administrateur : Tableau de bord personnalisé pour l'administrateur avec un aperçu des options disponibles.


  3. Gestion des Produits : Ajout, modification et suppression des bonbons du catalogue avec des fonctionnalités telles que la modification du prix, de la description et de l'image.
