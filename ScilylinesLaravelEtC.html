Application de Gestion de Flotte de Ferrys

Objectif :


Dans le cadre de ce projet, j'ai été chargé de développer une application web avec le framework Laravel pour l'entreprise SicilyLines. L'objectif était de mettre en place un intranet permettant de gérer sa flotte de ferrys.


Langages :

PHP (Laravel), HTML, CSS, JavaScript


Base de données :

MySQL


Fonctionnalités clés :


  1. Page d'Accueil : Mise en place d'un carousel et d'un bouton de connexion pour accéder à la liste des ferrys. L'accès à la liste est réservé aux utilisateurs connectés.
  2. Liste des Ferrys : Affichage de la liste des ferrys une fois connecté.
  3. Détails d'un Ferry : Implémentation du bouton "Voir" permettant d'accéder aux détails d'un ferry.
  4. Ajout et Suppression de Ferrys : Création d'un formulaire pour ajouter un ferry et d'un bouton pour le supprimer.
  5. Génération de Brochure PDF : Ajout d'un bouton permettant de générer une brochure PDF présentant les caractéristiques des ferrys réservés aux voyageurs.




Application de Gestion de Données Flotte de Ferrys

Objectif :


Ce projet consistait à développer une application en C# pour l'administrateur du portail internet, lui permettant de tester et de modifier les données de la base de données existante.


Langages :


C#


Base de données :

MySQL


Fonctionnalités clés :


  1. Interface Graphique : Création d'une interface graphique conviviale avec Visual Studio en C#, permettant à l'administrateur de gérer les données.
  2. Affichage des Secteurs et Liaisons : Affichage des secteurs dans une ListBox et affichage des liaisons correspondantes en sélectionnant un secteur.
  3. Opérations sur les Liaisons : Possibilité de supprimer une liaison, insérer une nouvelle liaison et modifier la durée d'une liaison.
  4. Architecture en Couches : Respect d'une architecture en couches pour assurer la maintenabilité et la facilité d'évolution du code.
